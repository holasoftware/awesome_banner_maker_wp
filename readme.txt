== Description ==
With this banners designer, you can create your owm images for your ads or for the featured image in your posts.

It's easy to use and at the same time powerful enough to create attractive banner designs.


== Main Features ==
- Easy to use
- Drag and drop layers
- Use text and image layers
- Choose between a background image or a gradient
- Import / Export
- Download the image or upload to the image gallery

== Frequently Asked Questions ==

= How to use this plugin? =
Just choose a background and then start adding layers. When you are done, save your design. You can in any moment convert the design to an image, and then upload the image to the gallery or download in local.
